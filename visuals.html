Visuals

In this step, we will install mods improving the overall look and feel of the game, without any drastic or performance demanding changes

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create separator.
  2. Name the separator Visuals.

Mostly Fixed FaceGen Tints

Installation instructions:

  • Main Files - FaceGen Tint Fixes for NV
Fixes the odd skin color you'll see on NPCs, including the player.

Muzzle Flash Light Fix - NVSE

Installation instructions:

  • Main Files - Muzzle Flash Light Fix
Fixes muzzle flash lights not being culled after shot, leading to game still using them for illumination.

Viewmodel Shading Fix - NVSE

Installation instructions:

  • Main Files - Viewmodel Shading Fix - NVSE
Fixes totally wrong specular lighting, and lack of muzzle flash lights on viewmodel.

Anniversary Anim Pack

Installation instructions:

  • Main Files - Anniversary Anim Pack
Overhauls most gun animations.

FNV Clean Animations

Installation instructions:

  • Main Files - FNV Clean Animations - Anniversary Edition
Another animation overhaul that goes well with the Anniversary Anim Pack.

After the installation of these two animation replacers, you'll need to reinstall Iron Sights Aligned mod in order to patch the new animations. To do so, follow these steps:

  1. Right-click Iron Sights Aligned under your list's Bug Fixes separator, then click Reinstall Mod.
    • If you cleared your downloads folder, you'll have to redownload the mod.
  2. You will be prompted with a FOMOD installer. Select following:
    1. Anniversary Anim Pack
    2. Yukichigai's Unofficial Patch
    3. Install
  3. Click Replace when prompted.
  4. Move Iron Sights Aligned under FNV Clean Animations (in the Visuals separator).

Smooth True Iron Sights Camera

Installation instructions:

  • Main Files - Smooth True Ironsights
Aim down sights without the camera snapping to place. Engine plugin, no esps or animation files - compatible with all animation mods.

B42 Weapon Inertia

Installation instructions:

  • Main Files - B42 Inertia V1.2
Gives weapons realistic inertia movement.

B42 Optics - ESPless

Installation instructions:

  • Main Files - B42 Optics
  • B42 Optics NV Patches
    • Main Files - B42 Optics NV Patches
      • You will be prompted with a FOMOD installer. Select following:
        1. Anniversary Anim Pack Fix
        2. Install
Replaces scopes with proper picture-in-picture ones.

NV Compatibility Skeleton

Installation instructions:

  • Optional Files - NVCS Vanilla Weights
Allows multiple mods that edit the skeleton mesh to work together, required for B42 Weapon Inertia.

High Resolution Water Fog - Water Aliasing Fix

Installation instructions:

  • Main Files - High Resolution Water Fog
Increases resolution of the water depth render target, significantly reducing aliasing.

Contrasted LOD Noise Texture

Installation instructions:

  • Main Files - Contrasted LOD Noise 1k
New LOD noise texture for Mid Frequency Details, helps blend far terrain with closer details.

Landscape Texture Improvements

Installation instructions:

  • Main Files - Landscape Texture Improvements
    • You will be prompted with a FOMOD installer. Select following:
      1. Next
      2. YUP Patch
      3. Install
Improves many aspects of the landscape, most notably reducing texture seams.

Wasted Rocks Normals

Installation instructions:

  • Main Files - First release.
Fixes the flipped lighting on the vanilla rock normal maps.

More Consistent Vanilla Rock Textures (Less Green Clifftops)

Installation instructions:

  • Main Files - More Consistent Vanilla Rock Textures
Makes the color of the canyon rubble and rock trim throughout most of the Mojave match better with the other meshes.

A Little More Lamplight

Installation instructions:

  • Main Files - A Little More Lamplight
Enhances the vanilla streetlights and adds new ones around Outer Vegas, Freeside and more.

Atmospheric Lighting Tweaks

Installation instructions:

  • Main Files - Atmospheric Lighting Tweaks - FNV - YUP
  • Vish's Patch Hub
    • Main Files - Atmospheric Lighting Tweaks - EVEM Patch (only if you installed Essential Vanilla Enhancements Merged)
Makes changes to Image Spaces and Lighting Templates to give interiors a more dynamic and atmospheric look.

Desert Natural Weathers

Installation instructions:

  • Main Files - Desert Natural Weathers NV
Major overhaul to all of the weathers for the base game and DLCs.

High Resolution Bloom NVSE

Installation instructions:

  • Main Files - High Resolution Bloom
Allows for customizable bloom resolution, making it possible to use resolutions other than vanilla's quarter one.

Cloud Upgrade NVSE

Installation instructions:

  • Main Files - Cloud Upgrade
Adds pseudo-animated clouds for horizons and panoramic cloud systems.

Interior Rain

Installation instructions:

  • Main Files - Interior Rain
Adds rain sounds to interiors.

3D Rain

Installation instructions:

  • Main Files - 3D Rain
Replace the ugly 2D rain with a new 3D version. Rain no longer looks weird when you look up or down.

Climate Control - Rain

Installation instructions:

  • Main Files - Climate Control - Rain
  • Optional Files - Climate Control - 3D Rain
Fixes rain not activating or deactivating correctly, as well as new sounds and textures.
