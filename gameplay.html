Gameplay

In this step, we will install mods aiming to improve the general gameplay

It is especially important to read the descriptions of these mods.

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create separator.
  2. Name the separator Gameplay.

JAM - Just Assorted Mods

Installation instructions:

  • Main Files - Just Assorted Mods
  • Viva New Vegas Resources
    • Main Files - JAM - VNV Configuration
  • JLM Grab Tweak
    • Main Files - JLM Grab Tweak
  • Clean Just Assorted Mods (JAM)
    • Main Files - Clean Just Assorted Mods
Highly configurable set of new modern features such as a quick loot menu, dynamic crosshair, sprinting, and more. Settings can be tweaked using the Mod Configuration Menu found in the in-game pause menu.
The Viva New Vegas preset for JAM only uses the following features:
  • Dynamic Crosshair - Modern crosshair that resizes in accordance to the weapon's spread.
  • Hit Marker - Crosshair markers signaling if you hit the target.
  • Hit Indicator - Arrows indicating the location of incoming damage.
  • Loot Menu - Fallout 4-style loot menu that opens whenever you look at a container.
  • Weapon Hweel - Quick-Use wheel menu.
Sprinting is disabled by default due to being replaced by Enhanced Movement.

Enhanced Movement

Installation instructions:

  • Main Files - Enhanced Movemenet
  • Viva New Vegas Resources
    • Main Files - Enhanced Movement - VNV Configuration
AIO movement expansion - featuring movement bug fixes, 1st person diagonal movement, enhanced sprint, enhanced sneak, enhanced leaning, animated prone framework, new animations, and more.
The Viva New Vegas preset for Enhanced Movement enables only fixes and the sprinting mechanic.

Immersive Recoil 2.0

Installation instructions:

  • Main Files - Immersive Recoil 2.4.1
Adds a visually smooth, dynamic and realistic recoil system.

Follower Tweaks

Installation instructions:

  • Main Files - Follower Tweaks
Reworks Boone, ED-E and Rexes' companion perks to be less intrusive/annoying.

Leveled DLC Delay

Installation instructions:

  • Main Files - Leveled DLC Delay
Delays DLC until you meet the required level or visit the respective location. Also includes DLC Hostility Fixes.

Better Caravan

Installation instructions:

  • Main Files - Better Caravan
Significantly improves the usability of the Caravan minigame. Optionally fixes AI bugs to make them a little smarter.

Cookable Grenades

Installation instructions:

  • Main Files - lStewieAl - Cookable Grenades
Holding a grenade before throwing it decreases its fuse.

Faster Sleep-Wait

Installation instructions:

  • Main Files - Faster Sleep Wait
Decreases the delay between hours while waiting or sleeping.

Auto-Save Manager

Installation instructions:

  • Main Files - Save Manager
Lightweight mod adding timed autosaves, save slots and incremental/full save hotkeys.

Bad Touch NVSE

Installation instructions:

  • Main Files - Bad Touch NVSE
  • Optional Files - Bad Touch NVSE - Minimalist Preset
Makes hazards such as fire and electricity actually deal damage.
